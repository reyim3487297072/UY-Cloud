  ##  应用介绍：
UY云是一款新的应用管理系统，是用来管理应用，小程序，网站等，UY云可以理解为是个第三方管理应用，目前已开发安卓及鸿蒙版，苹果版等正在开发中。
  UY云中可自定义管理应用，我们提供了用户管理，更新模块，通知等多个实用及常用模块功能，我们今后会尽力的优化及提供更多的云服务。
  ##  软件核心服务：
1-用户管理模块  
2-更新及公告模块  
3-应用通知模块  
4-HTML文档模块  
5-TEXT文档模块  
6-商城模块  
7-应用模块  
8-电影模块  
9-应用托管模块  
10-卡密模块  
11-云盘模块  
12-机器人模块  
13-资源模块  
14-论坛模块  
15-自定义API模块  
16-...  
 刚刚起步，会有不完善的地方，我们会向往为用户提供更多更良好的服务。
