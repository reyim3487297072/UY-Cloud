# UY-Cloud
UY云是新的第三方多功能，稳定的后台管理系统
  ##  应用介绍：
UY云是一款新的应用管理系统，是用来管理应用，小程序，网站等，UY云可以理解为是个第三方管理应用，目前已开发安卓及鸿蒙版，苹果版等正在开发中。
  UY云中可自定义管理应用，我们提供了用户管理，更新模块，通知等多个实用及常用模块功能，我们今后会尽力的优化及提供更多的云服务。
